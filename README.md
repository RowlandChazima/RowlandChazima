## About

I'm an 18-year-old self-taught developer based in Nairobi, currently deepening my skills in full-stack and AI engineering. I care about building software that solves real problems for real people — right now that means exploring how AI can meaningfully support teachers and students across Africa.

I hold a DELF B2 French certification and an NVIDIA GenAI/LLM certification, and I'm building under my own company, **Nuru Labs**.

My approach to engineering: understand *why* before writing *how*. Read the error before guessing the fix. Build the smallest working version before adding sophistication.
